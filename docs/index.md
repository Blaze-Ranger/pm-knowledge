## 关于本站

这是一个用 [VitePress](https://vitepress.dev/) 搭建的个人知识管理博客,目标是沉淀长期有价值的学习笔记与思考。

技术栈:VitePress + Vue 3 + TypeScript + pnpm + GitHub Actions + Giscus。
