# PRD 撰写清单

> 占位笔记 — 请用真实笔记替换。

## 一、需求背景

- [ ] 解决了谁的问题?
- [ ] 当前的痛点与机会成本是什么?
- [ ] 不做的代价是什么?

## 二、用户与场景

- [ ] 核心用户画像(年龄/职业/使用频次)
- [ ] 关键使用场景(时间/地点/触发条件)
- [ ] 用户的预期目标与情绪曲线

## 三、需求范围

- [ ] MVP(必须做)
- [ ] 增强(应该做)
- [ ] 锦上添花(可以做)
- [ ] 不做(明确排除)

## 四、方案设计

- [ ] 信息架构与流程图
- [ ] 关键页面线框
- [ ] 边界情况与异常流

## 五、衡量指标

- [ ] 北极星指标
- [ ] 过程指标(激活、转化、留存)
- [ ] 上线后的回顾计划
